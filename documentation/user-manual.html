Installation Guide

  1. Upload the downloaded zip file to your hosting server /public_html/ directory
  2. After complete upload then extract the uploaded zip file
  3. Make sure in project folder root index.php and .htaccess file have permission as 644
  4. Manually create a database from your server cpanel using MySql Database Wizard.
  5. Create a user for the database and link that user to the created database with all necessary permissions
  6. Open the file database.php from the directory application/config/database.php
  7. Fill up the information like "hostname", "username", "password" and "database" with your created database information
  8. Now from your server cpanel phpMyAdmin go to your database and import database.sql located in /documentation/database/database.sql
  9. Please check your site base url in application/config/config.php
  10. Run the project from any web browser like (http://www.yourdomain.com/).
  11. Enter your Purchase Code and submit.
  12. Now your are good to go!!
  13. Default Super Admin user credentials: Email: superadmin@gsms.com Password: 123456
  14. Good Luck!!!
  15. Thank you.

Step by Step Data Input Process:-

For smooth use of the software please follow the following data input process. These process will makes your software organized to data manipulation. You must follow these data input sequence.

  1. Login to System using default admin user credentials: Email: superadmin@gsms.com Password: 123456
  2. Create Academic Year as per your school requirements. Left Menu->Administrator->Academic Year
  3. Update General Setting and must set Academic Running Year. Left Menu->Setting->General Setting
  4. Update each User Type Role Permission as per your School requirements. Left Menu->Administrator->Role Permission->Role Permission Setting
  5. Create/Input Designations & Employees data as per your School requirements. Left Menu->Human Resources->Designation/Employees
  6. Create/Input Teachers data as per your School requirements. Left Menu->Teacher
  7. Create/Input Class data as per your School requirements. Left Menu->Class
  8. Create/Input Section data as per your Class & School requirements. Left Menu->Section->Class
  9. Create/Input Subject data as per your Class & School requirements. Left Menu->Subject->Class
  10. Create/Input Syllabus data as per your Class & School requirements. Left Menu->Syllabus->Class
  11. Create/Input Class Routine data as per your Class & School requirements. Left Menu->Class Routine->Class
  12. Create/Input Guardian data before input Student data. Left Menu->Guardian
  13. Create/Input Student data. Left Menu->Student->Student Class
  14. Create/Input Assignment data for student. Left Menu->Assignment->Class
  15. Create/Input Exam Grade data as per School requirements. Left Menu->Exam->Exam Grade
  16. Create/Input Exam Term data as per School requirements. Left Menu->Exam->Exam Term
  17. Create/Input Exam Schedule data. Left Menu->Exam Schedule->Class
  18. Create/Input Exam Suggestion data. Left Menu->Exam Suggestion->Class
  19. Create/Input Exam Attendance data. Left Menu->Exam Attendance
  20. Manage Exam Mark data. Left Menu->Exam Mark->Manage Mark
  21. Process Exam Term Result data. Left Menu->Exam Mark->Exam Term Result
  22. Process Exam Final Result data. Left Menu->Exam Mark->Exam Final Result
  23. View Exam Merit List data. Left Menu->Exam Mark->Exam Merit List
  24. Manage Exam Mark Sheet data. Left Menu->Exam Mark->Student Mark Sheet
  25. Process Student Result Card data. Left Menu->Exam Mark->Student Result Card
  26. Update SMS Setting data for sending SMS. Left Menu->Setting->SMS Setting
  27. Process Student Promotion to the next class after process all above data. Left Menu->Promotion
  28. Create/Input Certificate Type data. Left Menu->Certificate->Certificate Type
  29. Generate Certificate for a student. Left Menu->Generate Certificate Then filter and Generate a Certificate and print
  30. Create/Input Salary Grade data for Manage Payroll. Left Menu->Payroll->Salary Grade
  31. Create/Input/Process Salary Payment data for Employee/Teacher. Left Menu->Payroll->Salary Payment Then filter employee/teacher and put payment
  32. Manage Fee Type for Student Fee. Left Menu->Accounting->Fee Type
  33. Manage Fee Collection for Student Fee. Left Menu->Accounting->Fee Collection
  34. Manage Invoice for Student Fee. Left Menu->Accounting->Manage Invoice
  35. Create/Input Income Head data for Manage Income. Left Menu->Accounting->Income Head/Income
  36. Create/Input Expenditure Head data for Manage Expenditure. Left Menu->Accounting->Expenditure Head/Expenditure
  37. Create/Input Invoice data for manage/collect all kinds of Payment from student. Left Menu->Accounting->Create Invoice
  38. After Ctrate/Input Employee/ Teacher/ Guardian & Student data please activate then to login. Left Menu->Administrator->Manage User, Then filter User and activate them
  39. Good Luck!!!
  40. Thank you.
